> 数据来源：`PhysicsConstants.swift` → `TablePhysics` + `TableGeometry.swift`

---

## 坐标系说明

| 轴 | 方向 | 说明 |
|---|---|---|
| X | 台面长边方向 | 正 X = 右端（开球端反向端） |
| Y | 垂直向上 | 台面 Y = 0.80 m，球心 Y ≈ 0.8286 m |
| Z | 台面短边方向 | 正 Z = 上侧 |

- **台面中心** = (0, 0.80, 0)  
- **USDZ 模型**为 Z-up，加载时绕 X 轴旋转 −90° 转换到 Y-up。

**坐标计算方式：**
- 角袋 X 偏移 = innerLength/2 + cornerPocketRadius = 1.270 + 0.042 = **1.312 m**

> 袋口中心位于 playfield 边界**外侧**（超出有效击球区），因此 |X| > 1.270，|Z| > 0.635。

## 进袋检测关键数值

| 参数 | 角袋 | 中袋 |
|---|---|---|
| 袋口半径 | 42 mm | 43 mm |
| 进袋触发半径（= 袋口半径 − 球半径） | ≈ 13.4 mm | ≈ 14.4 mm |
| 进袋校验阈值（= 袋口半径 + 1.5 × 球半径） | ≈ 84.9 mm | ≈ 85.9 mm |

- 进袋检测在 **XZ 水平面**进行，忽略 Y 轴。
- 球心 XZ 距离 ≤ 触发半径时，触发四次方程候选事件。
- 事件分发时再次校验：XZ 距离 ≤ 校验阈值，否则视为数值误差丢弃。

## 球台坐标示意图（俯视，Y-up 坐标系）

```
Z+（上侧）
  +0.677 │ P1(-1.312,+0.677)        P3(+1.312,+0.677)
  +0.635 │ ┌──────┬──────────────────────┬──────┐
  +0.688 │ │      ║ P5(0,+0.688)         ║      │
         │ │      ║                      ║      │
         │ │  ←   ║    playfield         ║   →  │
         │ │  X-  ║   (2.540×1.270 m)    ║  X+  │
         │ │      ║                      ║      │
  -0.688 │ │      ║ P4(0,-0.688)         ║      │
  -0.635 │ └──────┴──────────────────────┴──────┘
```

- P0–P3：角袋（半径 42 mm）
- P4–P5：中袋（半径 43 mm）
- 台面有效区域：X ∈ [−1.270, +1.270]，Z ∈ [−0.635, +0.635]
