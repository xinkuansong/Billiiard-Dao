# 台球教练 (Billiard Trainer)

专业的中式八球训练 iOS App，帮助玩家系统化提升球技。

## 项目概述

- **目标用户**：台球初学者到进阶玩家
- **核心功能**：系统化课程教学 + 专项训练 + 数据统计
- **技术栈**：Swift 5.x + SwiftUI + SceneKit
- **最低系统**：iOS 16.0
- **支持设备**：iPhone

## 功能特性

### 📚 系统化课程
- 18节完整课程（入门 → 基础 → 进阶 → 高级）
- 覆盖瞄准、杆法、走位、翻袋、K球、颗星公式等

### 🎯 专项训练
- 7个训练场：瞄准、杆法、翻袋、K球、传球、解球、颗星
- 5星难度体系
- 实时进球率统计

### 📊 数据统计
- 进球率（按角度分类）
- 练习时长统计
- 技能雷达图

### 🎮 挑战模式
- 单球挑战
- 走位挑战
- 清台挑战

## 技术架构

```
BilliardTrainer/
├── App/                    # 应用入口
│   ├── BilliardTrainerApp.swift
│   └── ContentView.swift
├── Features/               # 功能模块
│   ├── Home/              # 首页
│   ├── Course/            # 课程
│   ├── Training/          # 训练
│   ├── Challenge/         # 挑战
│   ├── Statistics/        # 统计
│   └── Settings/          # 设置
├── Core/                   # 核心模块
│   ├── Physics/           # 物理引擎
│   ├── Aiming/            # 瞄准系统
│   ├── Camera/            # 相机控制
│   ├── Scene/             # SceneKit场景
│   └── Audio/             # 音效系统
├── Services/              # 服务层
├── Models/                # 数据模型
├── Resources/             # 资源文件
└── Utilities/             # 工具类
```

## 物理引擎

基于 SceneKit Physics，扩展实现：
- 旋转（塞）物理：上旋、下旋、侧旋
- 旋转衰减与传递
- 库边塞修正
- 专业级碰撞检测

## 颗星公式

支持：
- 一库颗星（翻袋）
- 两库颗星（K球）
- 三库颗星
- 塞修正、速度修正

## 运行项目

1. 克隆仓库
2. 用 Xcode 打开 `BilliardTrainer.xcodeproj`
3. 选择目标设备（iPhone 模拟器或真机）
4. 运行 (⌘+R)
