# Aiming（瞄准系统）- 验证与回归

> 对应 README：[README.md](./README.md)
> 文档最后更新：2026-02-27

## 影响面清单

改动本模块时，必须检查以下影响面：

- [ ] 调用方：`BilliardSceneView`（显示瞄准线/轨迹）、`TrainingViewModel`（瞄准辅助）
- [ ] 共享常量/状态：`BallPhysics`（球半径）、`CueBallStrike`（squirt 角）、`SeparationAngle`（分离角常量）、`TablePhysics`（台面尺寸）、`DiamondSystem`（颗星常量）
- [ ] UI 交互链：瞄准线显示、轨迹预览、袋口推荐、难度提示
- [ ] 持久化/数据映射：无（瞄准计算不持久化）
- [ ] 配置/开关：无

## 最小回归门禁

每次改动后必须验证（不可跳过）：

### 主路径验证

- [ ] **瞄准计算流程**：选择目标球和袋口 → 调用 `calculateAim()` → 返回正确的瞄准点、厚度、分离角 → 瞄准线显示正确
- [ ] **路径遮挡检测**：母球和目标球之间有障碍球 → `isPathOccluded()` 返回 `true` → 袋口分析排除被遮挡的袋口
- [ ] **可行袋口分析**：调用 `viablePockets()` → 返回按难度排序的袋口列表 → 推荐最容易的袋口

### 相邻流程验证（至少 2 个）

- [ ] **分离角计算**：不同厚度和杆法组合 → `calculateSeparationAngle()` 返回正确的分离角 → 高杆分离角减小，低杆分离角增大
- [ ] **Squirt 修正**：使用左右塞 → `calculateAim()` 应用 squirt 修正 → 瞄准方向正确偏移
- [ ] **颗星公式计算**：一库翻袋 → `calculateOneRailBank()` 返回正确的颗星位置 → 翻袋路径正确
- [ ] **走位预测**：调用 `predictCueBallPosition()` → 返回母球预计停止位置 → 位置合理（在台面范围内）

## 可观测性

- 日志前缀：`[Aiming]`
- 关键观测点：
  - `AimResult.thickness`：厚度值（应在 [0, 1] 范围内）
  - `AimResult.separationAngle`：分离角（应在 [0, 180] 度范围内）
  - `AimResult.difficulty`：难度评分（应在 [1, 5] 范围内）
  - `DiamondResult.firstRailDiamond.number`：第一库颗星编号（应在有效范围内）
- 可视化开关：瞄准线显示、轨迹预览（在 `BilliardSceneView` 中控制）

## 常见问题排查

| 症状 | 可能原因 | 排查路径 |
|------|----------|----------|
| 瞄准点不准确 | `BallPhysics.radius` 值错误或 squirt 修正未应用 | 检查球半径常量、检查 `spinX` 参数 |
| 分离角计算错误 | `SeparationAngle` 常量值错误或杆法修正未应用 | 检查分离角常量、检查 `spinY` 参数 |
| 路径遮挡检测失效 | 障碍球列表为空或检测逻辑错误 | 检查 `obstacles` 参数、检查投影计算 |
| 颗星公式计算返回 nil | 位置超出台面范围或边判断错误 | 检查输入位置、检查 `nearestEdge()` 逻辑 |
| 翻袋计算不准确 | 镜像位置计算错误或库边交点计算错误 | 检查镜像计算、检查交点计算 |

## 未覆盖风险与待补测项

| 风险描述 | 等级 | 建议补测方式 |
|----------|------|-------------|
| 极端角度（接近 90°）时厚度计算可能不准确 | 低 | 添加极端角度测试用例 |
| 多球重叠时路径遮挡检测可能不准确 | 低 | 添加多球重叠测试用例 |
| 三库公式简化实现可能不够准确 | 中 | 对比标准三库公式，必要时改进实现 |
| Squirt 角计算依赖 `CueBallStrike` 模块，该模块变更可能影响瞄准精度 | 中 | 添加 squirt 角回归测试 |
| 走位预测不考虑库边反弹，可能不准确 | 低 | 添加库边反弹预测（如需要） |
