> 对应 README：[README.md](./README.md)

## 影响面清单

改动本模块时，必须检查以下影响面：

- [ ] 调用方：`ContentView`（主内容视图，根据 `isFirstLaunch` 显示引导页或主界面）、`MainTabView`（主界面 Tab 导航，包含 `HomeView`）
- [ ] 共享常量/状态：`AppState`（用户数据、首次启动标志、学习进度、今日统计）
- [ ] UI 交互链：应用启动 → 引导页（首次） → 首页 → 快捷入口导航 → 其他模块
- [ ] 持久化/数据映射：`AppState.isFirstLaunch`（UserDefaults 或 SwiftData），用户数据持久化（等级、经验值、学习进度、统计数据）
- [ ] 配置/开关：无特殊配置项

## 最小回归门禁

每次改动后必须验证（不可跳过）：

### 主路径验证

- [ ] **首次启动流程**：删除应用重新安装 → 启动应用 → 验证显示引导页 → 滑动查看3页内容 → 点击"开始使用" → 验证引导页消失 → 验证显示主界面
- [ ] **首页显示流程**：启动应用（非首次） → 验证显示主界面 → 验证用户信息卡片显示 → 验证快捷入口显示 → 验证学习进度显示 → 验证今日统计显示

### 相邻流程验证（至少 2 个）

- [ ] **引导页跳过流程**：首次启动 → 显示引导页 → 点击"跳过" → 验证引导页消失 → 验证显示主界面 → 验证不再显示引导页（重启应用）
- [ ] **快捷入口导航流程**：首页 → 点击"自由练习" → 验证打开 `FreePlayView` → 返回首页 → 验证首页状态正常
- [ ] **数据更新流程**：完成一次训练 → 返回首页 → 验证今日统计更新（练习时长、进球数、进球率） → 验证用户经验值更新（如适用）
- [ ] **非首次启动流程**：完成首次启动引导 → 重启应用 → 验证直接显示主界面 → 验证不显示引导页

## 可观测性

- 日志前缀：`[Home]`
- 关键观测点：
  - `OnboardingView` 页面切换：记录当前页面索引，记录"下一步"和"跳过"操作
  - `HomeView` 数据加载：记录从 `AppState` 读取的用户数据（等级、经验值、学习进度、统计数据）
  - 快捷入口导航：记录导航目标（继续学习、快速练习、自由练习），记录导航成功/失败
- 可视化开关：首页UI显示用户信息和统计数据，可用于调试数据绑定

## 常见问题排查

| 症状 | 可能原因 | 排查路径 |
|------|----------|----------|
| 引导页不显示（首次启动） | `AppState.isFirstLaunch` 未正确设置或检查逻辑错误 | 检查 `ContentView` 中 `isFirstLaunch` 检查逻辑，检查 `AppState` 初始化，验证 UserDefaults 或 SwiftData 存储 |
| 引导页重复显示（非首次启动） | `AppState.completeOnboarding()` 未正确设置 `isFirstLaunch = false` | 检查 `OnboardingView` 中完成逻辑，检查 `AppState.completeOnboarding()` 实现，验证持久化存储 |
| 首页数据不显示 | `AppState` 数据缺失或未正确注入 | 检查 `HomeView` 中 `@EnvironmentObject` 注入，检查 `AppState` 初始化，验证数据读取逻辑 |
| 快捷入口导航失败 | 目标视图不存在或初始化失败 | 检查导航目标视图（`FreePlayView`、训练列表、课程列表）是否存在，检查导入语句，查看错误日志 |
| 统计数据不更新 | `AppState` 统计数据未更新或视图未响应 | 检查统计数据更新逻辑（应在训练/游戏完成后更新），检查 `AppState` 状态变更通知，验证视图更新 |
