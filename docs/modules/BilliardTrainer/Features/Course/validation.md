## 影响面清单

改动本模块时，必须检查以下影响面：

- [ ] 调用方：ContentView（TabView 导航）、HomeView（可能显示课程进度）
- [ ] 共享常量/状态：Course 静态数据（freeCourses、basicCourses 等）
- [ ] UI 交互链：课程卡片点击（当前无导航，需验证点击响应）
- [ ] 持久化/数据映射：无（当前为静态数据）
- [ ] 配置/开关：无

## 最小回归门禁

每次改动后必须验证（不可跳过）：

### 主路径验证

- [ ] 打开课程 Tab，验证四个课程分组（入门、基础、进阶、高级）正常显示
- [ ] 验证课程卡片显示正确：编号（L1-L18）、标题、描述、时长、状态图标
- [ ] 验证锁定状态：免费课程未锁定（绿色标签），付费课程锁定（橙色标签 + 锁定图标）

### 相邻流程验证（至少 2 个）

- [ ] 从首页点击课程入口，验证导航到课程列表正常
- [ ] 验证课程卡片点击响应（如有导航逻辑）

## 常见问题排查

| 症状 | 可能原因 | 排查路径 |
|------|----------|----------|
| 课程列表不显示 | 导航路径错误 | 检查 ContentView TabView 配置 |
| 课程卡片状态错误 | 锁定状态硬编码不一致 | 检查 CourseSection isLocked 参数 |
| 课程数据缺失 | 静态数据未初始化 | 检查 Course 静态常量定义 |

## 未覆盖风险与待补测项

| 风险描述 | 等级 | 建议补测方式 |
