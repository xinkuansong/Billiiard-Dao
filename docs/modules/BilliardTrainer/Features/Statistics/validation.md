## 影响面清单

改动本模块时，必须检查以下影响面：

- [ ] 调用方：ContentView（TabView 导航）
- [ ] 共享常量/状态：UserStatistics SwiftData 模型（当前未集成，待验证）
- [ ] UI 交互链：统计卡片展示、数据格式化显示
- [ ] 持久化/数据映射：UserStatistics 模型字段映射（totalPracticeTime、totalPocketed、overallAccuracy 等）
- [ ] 配置/开关：无

## 最小回归门禁

每次改动后必须验证（不可跳过）：

### 主路径验证

- [ ] 打开统计 Tab，验证四个统计卡片正常显示（总览、进球率、练习时长、技能雷达）
- [ ] 验证总览卡片数据：总练习时长、总进球数、平均进球率格式正确
- [ ] 验证进球率卡片：直球、30°、45°、60° 角度球进球率显示，颜色阈值正确（绿色 ≥70%，橙色 ≥50%，红色 <50%）

### 相邻流程验证（至少 2 个）

- [ ] 从首页点击统计入口，验证导航到统计页面正常
- [ ] 验证练习时长柱状图：周一到周日数据正常显示，今日高亮显示

## 常见问题排查

| 症状 | 可能原因 | 排查路径 |
|------|----------|----------|
| 统计数据不显示 | UserStatistics 查询失败 | 检查 @Query 或数据模型配置 |
| 进球率计算错误 | 除零或数据格式不一致 | 检查 guard 保护和数据格式 |
| 柱状图高度异常 | 高度系数或数据单位错误 | 检查 PracticeTimeCard weekData 和高度计算 |
